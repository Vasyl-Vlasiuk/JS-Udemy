Если я уже запушил проект без файла .gitignore, то что делать, чтобы тот, кто будет пуллить его, не качал эту большую папку?



Добрый день, на данном этапе вы можете удалить реппозиторий и залить еще раз,

Или же добавьте файл в .gitignore

git rm --cached --ignore-unmatch название файла - удаляем файл только из репозитория, и физически файл сохранится на диске

git commit -am "Message"

git push origin {branch name}
